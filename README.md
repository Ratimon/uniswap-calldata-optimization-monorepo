# uniswap-calldata-optimization-monorepo


## Quickstart

### Smart Contracts

1. Spinning up environment

```sh
cd contracts
pnpm start
```

2. deploy contracts to localnetwork

```sh
pnpm deploy_all
```

3. kill environment when not needed

```sh
cd contracts
pnpm stop
```

### Frontend

```sh
cd web
pnpm dev
```
